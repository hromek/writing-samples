Historic Weather Data

Prime Weather provides functionality for accessing historic weather data on an hourly basis for a single day and on a daily basis for range of days. The service also provides monthly data averages.

Creating a History Object

The History class provides methods for retrieving historic weather data. The example below creates a new History object using a previously created Connection object (see Connecting to Prime Weather).

//Create a History object.
History history = new History(connection);

Getting Hourly Historic Weather Data

Use the getHourlyConditions() method of a History object to retrieve weather data for each hour of a single day. The method returns the data as a HashMap of Time and Conditions (<String, String>). The example below gets the weather data for January 1, 1962 and prints the low temperature values.

//Get historic data for a single day and print the low temperature values.
java.time.LocalDate date = java.time.LocalDate.parse("1962-01-01");
java.util.HashMap<java.time.LocalTime, Conditions> historicHourlyConditions = history.getHourlyConditions(date);
historicHourlyConditions.forEach((time, conditions) 
    -> System.out.println("Time:" + time + " Low Temperature" + conditions.get("Low Temperature"))

Getting Daily Historic Weather Data

Use the getDailyConditions() method of a History object to retrieve weather data for each hour of a single day. The method returns data as a HashMap of Time and Conditions (<String, String>). The example below gets the weather data for January 1, 1962 and prints the high temperature values.

//Get historic data for a single day and print the high temperature values.
java.time.LocalDate startDate = java.time.LocalDate.parse("2002-01-01");
java.time.LocalDate endDate = java.time.LocalDate.parse("2002-01-31");
java.util.HashMap<java.time.LocalDate, Conditions> historicDailyConditions = 
        history.getDailyConditions(startDate, endDate);
historicDailyConditions.forEach((historicDate, conditions) 
    -> System.out.println("Date:" + historicDate + " High Temperature" + conditions.get("High Temperature"))
);

Getting Monthly Averages

Use the getAverageConditions() method of a History object to retrieve average weather conditions for each month of a single year. The method returns the data as a HashMap of Month and Conditions. The example below gets the average conditions for 2003 and prints the average high temperature values.

//Get the average conditions for the year 2003 and print the high average temperature values.
int year = 2003;
java.util.HashMap<java.time.Month, Conditions> averageConditions = history.getAverageConditions(year);
averageConditions.forEach((month, conditions)
    -> System.out.println("Month: " + month + " Average High Temperature: " + conditions.get("Average High Temperature"))
);
