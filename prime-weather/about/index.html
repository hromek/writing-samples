About this Writing Sample

Java and NetBeans

Prime Weather is a ficticious cloud service application contrived for the purpose of supporting this writing sample. I used the NetBeans Integrated Developent Environment (IDE) to create skeleton classes for the API and to write the code examples. Using this approach enabled me to ensure that all code examples compiled correctly.

Markdown and MkDocs

I used Markdown and MkDocs to write this documentation example. MkDocs is a static website generator that converts Markdown text to HTML web pages. NetBeans provides a Markdown plugin, so I also used the NetBeans IDE as the editor for Markdown text. MkDocs ships with several starndard themes and has a framework for developing custom themes. I used one of the standard themes with just a few tweaks to the HTML and CSS.
