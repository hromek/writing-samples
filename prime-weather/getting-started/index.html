Getting Started

Create a Prime Weather Service Account

An Account Key is required to access the Prime Weather system. Obtain a unique account key by visiting the Prime Weather website and creating a new account. The website will provide your unique account key at the end of the registration process. You can obtain your key from the website at any time by logging into your account and navigating to the Account Information page.

Download the Java API

After your account is created, download the Java API JAR file from the Download page of the Prime Weather website. The JAR file contains the classes required for connecting to Prime Weather and accessing weather data. Consult the documentation for your development environment for instructions on using classes from JAR files.
