What is Prime Weather?

Prime Weather is a cloud-based application that provides detailed weather forecasts for the entire world. Using proprietary algorithms, Prime Weather combines data from 17 weather models to deliver current conditions and the most accurate forecasts available. The application also provides detailed historic weather data back to 1849 for the United States.

Java API

The Prime Weather Java API allows developers to quickly integrate weather data capabilities into their applications and provides functionality to easily retrieve current weather conditions, forecasted future conditions, and historic data. The API is distributed as a single JAR file with several supported versions available. Prime Weather recommends using the latest stable version for production implementations.
