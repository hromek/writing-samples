Weather Forecasts

The Prime Weather Java API provides access to current and forecasted weather conditions for any location on Earth. The forecast data includes hourly conditions for the next 48 hours and daily conditions for the next ten days. The API also provides access to active weather alerts and radar images.

Creating a Forecast Object

The Forecast class provides methods for retrieving current and forecasted weather data. The example below creates a new Forecast object using a previously created Connection object (see Connecting to Prime Weather) and changes the geographic location.

//Create a Forecast object.
Forecast forecast = new Forecast(connection);

//Change the location to Key West, FL.
double KeyWestLatitude = 24.5551;
double KeyWestLogitude = -81.7800;
connection.setLocation(KeyWestLatitude, KeyWestLogitude);

Getting the Current Weather Conditions

Use the getCurrentConditions() method of a Forecast object to retrieve the current weather conditions. The example below creates a Forecast object and prints the current temperature and humidity. 

//Get the Current weather conditions and print the Temperature and Humidity.
Conditions currentConditions = forecast.getCurrentConditions();
System.out.println("Current Temperature: "+ currentConditions.get("Temperature"));
System.out.println("Current Humidity: " + currentConditions.get("Humidity"));

Getting Hourly Forecasts

Use the getHourlyForecast() method of a Forecast object to retrieve weather data for the next 48 hours. The method returns the data as a HashMap of Time and Conditions. The example below gets hourly weather data and prints the all Time and Temperature values. 

//Get the Hourly forecast and print the Time and Temperature values.
java.util.HashMap<java.time.LocalTime, Conditions> hourlyForecast = forecast.getHourlyForecast();
hourlyForecast.forEach((time, conditions) 
    -> System.out.println("Time:" + time + " Temperature" + conditions.get("Temperature"))
);

Getting Daily Forecasts

Use the getDailyForecast() method of a Forecast object to retrieve weather data for the next 10 days. The method returns data as a HashMap of Date and Conditions. The example below gets daily weather data and prints all Date and High Temperature values. 

//Get the Daily forecast and print the Date and Temperature values.
java.util.HashMap<java.time.LocalDate, Conditions> DailyForecast = forecast.getDailyForecast();
DailyForecast.forEach((date, conditions) 
    -> System.out.println("Date:" + date + " High Temperature" + conditions.get("High Temperature"))
);

Getting Weather Alerts

Use the getAlerts() method of a Forecast object to retrieve an array of Alert objects representing all currently active alerts for the current location. Each Alert object consists of an Alert Summary and a Discussion, which provides detailed information about severe weather conditions. The example below gets all current weather alerts and prints the alert summaries.

//Get the alerts and print the summaries.
java.util.ArrayList<Alert> alerts = forecast.getAlerts();
alerts.forEach((alert)
    -> System.out.println("Alert Summary: " + alert.getSummary()));

Getting Radar Images

Use the getRadarImage() method of a Forecast object to retrieve dynamically generated radar images. The range parameter specifies the radius of the radar image coverage area (be sure to specify the range in units that match the units of measure currently configured on the Connection object). The animated parameter determines whether the method returns a static PNG or an animated GIF image. The animated image displays radar images for the past four hours in 10 minute intervals. The example below retrieves an animated radar image for the area covered by a 100 kilometer radius.

//Get the animated 100km radar image.
int range = 100; //kilometers
boolean animated = true;
byte[] radar100 = forecast.getRadarImage(range, animated);
